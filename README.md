# 🪨📄✂️ Rock Paper Scissors Game

A simple and fun **Rock Paper Scissors** game built using HTML, CSS, and JavaScript.

## 🎮 Features
- Player vs Computer mode
- Randomized computer choices
- Win/lose/draw detection
- Simple and clean UI
